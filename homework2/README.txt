I file dell'homework sono organizzati secondo la seguente struttura:
- html/ ---> directory che contiene i file html del sito web;
- css/ ----> directory che contiene i file css del sito web;
- img/ ----> directory che contiene le immagini sfruttate per la realizzazione del sito web.
Il file di partenza per avviare il sito web e' html/index.html .

L'obiettivo dell'homework e' la realizzazione di un sito web riguardo un torneo calcistico ad 8 squadre
denominato Champions League. Il torneo e' un campionato che prevede girone d'andata e ritorno.
Le funzionalita' offerte dal sito web sono:
1) visualizzazione della classifica del torneo, in accordo con le partite memorizzate;
2) visualizzazione del resoconto partite disputate;
3) inserimento di una nuova partita;
4) visualizzazione delle squadre partecipanti e riferimento al loro sito web ufficiale.

Si cerca quindi di far utilizzo dei principali elementi di XHTML: div, form, input ... in modo da 
organizzare le varie pagine in sezioni, avendo la possibilta' di gestire l'interazione con l'utente del sito.
Lo stile viene implementato grazie alle regole CSS cercando di gestire i fattori estetici (colore, sfondo ...) e
il posizionamento dei vari oggetti all'interno delle pagine.

Il naturale prosieguo di questo homework e' l'implementazione effettiva dell'inserimento di una nuova partita
con conseguente aggiornamento della classifica e del resoconto partite.
